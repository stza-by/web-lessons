Свойство dispaly

Мы можем управлять типом элемента (строчный или блочный) с помощью css свойства display

Тег p является блочным элементом, но с помощью свойства display: inline можем сделать его строчным

Теперь тег p будет вести себя как строчный (не имеет вертикальных отступов, не имеет переносов на новую строку)
